Estudos Unificados


Esse pequeno projeto visa juntar em um único local alguns aprendizados (principalmente de front-end)

Não há nada complexo, visto que é algo inicial ainda. De toda forma foi divertido fazer e espero que goste :)

Em caso de novos aprendizados nessa área, pretendo adicionar novas funcionalidades nesse projeto

Aqui usei conhecimentos de html, css e js. Também usei git para eviar para o github qualquer atualização.

A maior parte dos conhecimentos aqui registrados foram aprendidos na Alura, no qual espero avançar mais um pouco nesse vasto mundo e poder entregar cada vez mais!
